# 진행상황 한글 표시 안내

실행 시 터미널·노트북에 **한글로** 진행상황이 표시됩니다.

## 표시되는 한글 메시지 예시

### run_all.py / 파이프라인
- `[진행 1/2] 데이터셋 구축 중... (Vital 파일 → 특성·라벨 추출)`
- `[진행 1/2] 데이터셋 이미 있음 → 구축 생략`
- `[진행 2/2] 모델 학습 중... (PyTorch CUDA)`
- `[완료] 전체 파이프라인 실행이 끝났습니다.`

### 데이터셋 구축 (build_dataset)
- `[진행상황] 데이터셋 구축 시작 (총 N건 케이스 예정)`
- `[1/2] 케이스 처리 중` (tqdm 진행률)
- `[진행상황] 데이터셋 구축 완료 — N행 저장: ...`
- `[과금 방지] N분 도달 - 저장 후 중단` (한도 도달 시)

### 모델 학습 (train_model)
- `[진행상황] 학습 장치: cuda | train N건, test N건`
- `[2/2] 모델 학습 중` (tqdm, 단위: 배치)
- `[진행상황] 최종 모델 저장 완료 -> ...`
- `[진행상황] 평가 중...`
- `[결과] 분류 성능 (한글)` + `저혈압 없음` / `저혈압` 라벨
- `혼동 행렬:` (Confusion matrix)

### 주피터 노트북 (hypotension_pipeline.ipynb)
- 위와 동일한 한글 메시지가 셀 실행 시 출력됩니다.

## 자동 실행 방법

1. **Cursor / VS Code**: `Ctrl+Shift+B` → **"2. 전체 파이프라인 실행 (run_all)"** 또는 **"3. 주피터 노트북 자동 실행"**
2. **탐색기**: `run.bat` 또는 `run_notebook.bat` 더블클릭
3. **노트북**: `hypotension_pipeline.ipynb` 열고 **Run** → **Run All**

위 방법으로 실행하면 진행상황이 한글로 표시됩니다.
