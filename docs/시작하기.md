# 시작하기 — 한 번에 진행

**GitHub**: [VitalDB-Hypotension-Prediction](https://github.com/heuiseung/VitalDB-Hypotension-Prediction)

## 1. 설정 검증 (선택)

```
python check_setup.py
```

또는 Cursor/VS Code: **Ctrl+Shift+P** → **Tasks: Run Task** → **설정 검증 (check_setup)**

- 데이터 경로·케이스 수·vital 파일 샘플이 한글로 출력됩니다.
- 오류가 있으면 메시지로 원인을 확인한 뒤 경로를 수정하세요.

## 2. 파이프라인 실행 (한 번에)

**방법 A — 스크립트**

```
run.bat
```

또는 PowerShell: `.\run.ps1`  
→ 패키지 설치 후 데이터셋 구축(100건) → 학습까지 자동 실행.

**방법 B — Cursor / VS Code**

1. **Python 인터프리터 선택**: Ctrl+Shift+P → **Python: Select Interpreter** → Python 3.12/3.14
2. **Ctrl+Shift+B** → **"2. 전체 파이프라인 실행 (run_all)"** 선택

**방법 C — 주피터 노트북**

- **hypotension_pipeline.ipynb** 열기 → **Run** → **Run All**
- 또는 `run_notebook.bat` 더블클릭

## 3. GitHub 저장

- **push_to_github.bat** 더블클릭 또는 **Tasks** → **Git: GitHub에 자동 저장 (커밋+푸시)**

---

진행상황은 한글로 표시됩니다. 과금 방지(최대 시간/스텝) 도달 시 자동 저장 후 중단됩니다.
